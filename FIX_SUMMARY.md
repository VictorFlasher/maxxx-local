# Исправления проблем мессенджера

## 1. Проблема с внешним ключом (ForeignKeyViolation)

**Проблема:** При удалении сообщений или выходе из чата возникала ошибка:
```
UPDATE или DELETE в таблице "messages" нарушает ограничение внешнего ключа 
"last_read_messages_last_read_message_id_fkey"
```

**Решение:** Создан SQL-файл для исправления внешнего ключа с `ON DELETE CASCADE`:
```bash
psql -U postgres -d postgres -f /workspace/fix_fk.sql
```

Это позволит автоматически удалять записи из `last_read_messages` при удалении связанных сообщений.

---

## 2. Ложный alert "Не удалось открыть чат"

**Проблема:** При каждом открытии чата показывался alert об ошибке, даже если чат успешно загружался.

**Решение:** Убран `alert()` из функции `openChat()` в `/workspace/templates/chat.html`. Теперь ошибки только логируются в консоль.

---

## 3. WebSocket уведомления при редактировании/удалении сообщений

**Проблема:** При редактировании или удалении сообщения изменения не отображались у других пользователей без перезагрузки страницы.

**Решение:** 
- Убрано условие `if background_tasks` из функций `edit_message()` и `delete_message()` в `/workspace/app/routes/chat.py`
- Теперь WebSocket уведомления отправляются всегда сразу после коммита в БД

---

## 4. Выход из группового чата

**Проблема:** Ошибка FK при выходе из группового чата.

**Решение:** Функция `remove_user_from_group_chat()` уже содержит код для удаления записей из `last_read_messages` перед удалением участника. После применения `fix_fk.sql` проблема будет решена полностью.

---

## 5. Бан пользователей при авторизации

**Проблема:** Забаненные пользователи могли войти в систему, но не видели чаты.

**Решение:** Проверка бана добавлена на этапе аутентификации. Забаненные пользователи получают ошибку 403 с сообщением "Ваш аккаунт заблокирован".

---

## Как применить исправления:

1. **Применить SQL-фикс для внешнего ключа:**
   ```bash
   psql -U postgres -d postgres -f /workspace/fix_fk.sql
   ```

2. **Перезапустить сервер:**
   ```bash
   C-c  # остановить текущий сервер
   uvicorn main:app --reload
   ```

3. **Проверить работу:**
   - Открыть чат → не должно быть alert
   - Отправить сообщение → должно появиться у всех участников сразу
   - Редактировать сообщение → должно обновиться у всех
   - Удалить сообщение → должно исчезнуть у всех
   - Покинуть чат → должно работать без ошибок
   - Войти как забаненный пользователь → должно выдавать ошибку

---
